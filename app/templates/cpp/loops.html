Switch Statement

  • The switch statement tests a variable against a list of values, which are called cases, to determine whether it is equal to any of them.
  • default runs when none of the cases are true.
  • Syntax:
    switch (expression) {
    case value1:
        //Statement
        break;
    case value2:
        //Statement
        break;
    ...
    default:
        //Statement	
}
