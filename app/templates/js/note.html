SetInterval

  • It calls a function or evaluates an expression at specified intervals(in milliseconds).
  • It will contine calling the function until clearInterval() is called or the window is closed.
    E.g.
    function myAlert(){
    alert("Hi");
}
setInterval(myAlert, 3000);
//calls myAlert function every 3 seconds
    function myAlert(){
    alert("Hi");
}
setInterval(myAlert, 3000);
//calls myAlert function every 3 seconds

Form validation

Sample:

let validate = () => {
    if(document.myform.Email.value == ""){
        alert("Please provide your Email ID")
        document.myform.Email.focus()
        return false
    } else if(document.myform.Password.value == ""){
        alert("Please provide your Password")
        document.myform.Password.focus()
        return false
    } else {
        return true
    }
}
