• An exception is a way of handling an error that might occur when your program is running.
  • In order to throw an exception, you can use error numbers. E.g. throw 99;.
  • To handle the error, you need to catch the error (Which is the error number).
  • This is done using the catch keyword and enter a block of code to run if an error occurs.
try {
    int num1 = 55;
    int num2 = 0;
    
    if(num1 > num2) {
        throw 0;
    }
} catch(int x) {
    cout << "Math error: cannot divide by " << x << endl;
}
